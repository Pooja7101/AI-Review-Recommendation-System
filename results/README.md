Results

* Positive reviews
* Negative reviews
* Neutral reviews
* Sentiment distribution statistics

Aspect-Based Sentiment Analysis (ABSA)

* Extracted product aspects
* Aspect-wise sentiment scores

Recommendation Results

* Personalized product recommendations
* Similar product suggestions

Advertisement Generation

* AI-generated product advertisements using FLAN-T5

Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

 Visualizations

* Sentiment distribution charts
* Aspect frequency plots
* Recommendation performance graphs

Generated result files, reports, and visualizations can be stored in this folder.
